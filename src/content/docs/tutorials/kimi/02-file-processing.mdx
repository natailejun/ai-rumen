## 本教程目标

学会用 Kimi 处理各种文件。

---

## 支持的文件类型

| 类型 | 格式 |
|------|------|
| 文档 | PDF、Word (.docx)、TXT |
| 表格 | Excel (.xlsx) |
| 演示 | PPT (.pptx) |
| 图片 | JPG、PNG（提取图中文字） |
| 网页 | 直接粘贴 URL |

---

## 实操场景

### 场景 1：总结 PDF 报告

上传一个 PDF → 输入"总结这份报告的 3 个核心观点"

### 场景 2：分析 Excel 数据

上传 Excel → "分析这个季度的销售趋势，列出前 5 名产品"

### 场景 3：翻译合同

上传 PDF → "把这 3 页合同翻译成中文，专业术语保持准确"

### 场景 4：对比多个文档

上传两个文档 → "对比这两份方案，列出各自的优缺点"

---

## 提示

Kimi 的 200K 上下文意味着你可以**一次性上传整本书**让它分析。试试上传一本你读过的小说，让它写书评。
