# 📄 Kimi 从入门到精通

**Kimi** 是月之暗面（Moonshot AI）开发的中文 AI 助手，以**超长文本处理**闻名——一次能处理 200K 字（相当于一本《三体》的长度）。

手机号注册，完全免费，国内直接访问。

### [📂 文件处理与长文阅读](/tutorials/kimi/02-file-processing)
