# 🎨 AI 绘画入门 — 用 AI 画出你的想象

你有没有想过，只要动动嘴（打打字），就能画出精美的图片？

**AI 绘画**（也叫 AI 生成图片、文生图）就是这样一种神奇的技术——你用文字描述想要的画面，AI 就能帮你画出来。不需要任何美术基础，不需要学 Photoshop，甚至不需要会画画！

目前最流行的两个 AI 绘画工具是：

- 🟢 **DALL·E**（读作"达利"）—— 由 OpenAI 开发，**免费**集成在 ChatGPT 里，零门槛
- 🟣 **Midjourney**（简称 MJ）—— 付费工具，通过 Discord 使用，画质公认最强

## DALL·E vs Midjourney 对比

该选哪个？看完这个表格你就清楚了：

| 对比项 | DALL·E（ChatGPT） | Midjourney |
|--------|:---:|:---:|
| 💰 **价格** | 免费（有次数限制） | $10/月起 |
| 🎯 **上手难度** | ⭐ 非常简单 | ⭐⭐⭐ 需要学 Discord |
| 🖼️ **画质** | ⭐⭐⭐⭐ 很好 | ⭐⭐⭐⭐⭐ 最强 |
| 🌐 **访问方式** | ChatGPT 网页/APP | Discord（需科学上网） |
| 📱 **手机可用** | ✅ | ✅（通过 Discord APP） |
| 🎨 **艺术风格** | 多样，写实也不错 | 艺术感极强，细节丰富 |
| 🇨🇳 **国内可用** | ⚠️ 需科学上网 | ⚠️ 需科学上网 |
| 👉 **最适合** | 新手入门、日常使用 | 追求极致画质、专业创作 |

> 💡 **建议**：如果你是零基础新手，先从 DALL·E（教程 01）开始，完全免费，上手超快。等你对 AI 绘画有了基本了解后，再尝试 Midjourney，感受顶级画质的震撼。

## AI 绘画能用来做什么？

很多人以为 AI 画画只是"玩具"，其实它的用途非常广：

| 场景 | 示例 |
|------|------|
| 🖼️ **社交媒体配图** | 给公众号文章、小红书帖子配原创插图 |
| 📱 **头像和壁纸** | 生成独一无二的个人头像、手机壁纸 |
| 🎁 **个性化礼物** | 把朋友的照片变成艺术画，制作明信片 |
| 📊 **PPT 和演示文稿** | 快速生成专业的配图，告别素材网站 |
| 🏠 **装修效果图** | 描述你理想中的房间，AI 帮你"装修" |
| 👕 **图案设计** | T恤图案、手机壳图案、Logo 设计初稿 |
| 📖 **儿童绘本** | 给故事配上精美插画 |
| 🎮 **游戏概念图** | 快速出游戏场景和角色概念设计 |
