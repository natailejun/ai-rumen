学完这篇，你会拥有：

- ✅ 知道什么是 DALL·E，以及它和 ChatGPT 的关系

## 第一步：了解 DALL·E 是什么

**DALL·E**（读作"达利"，致敬超现实主义画家达利）是 OpenAI 开发的 AI 绘画模型。

简单来说：

| 概念 | 解释 |
|------|------|
| 🤖 **它是什么** | 一个能理解文字并画出图片的 AI |
| 💬 **怎么用** | 在 ChatGPT 里直接用中文描述你想要的画面 |
| 💰 **收费吗** | ChatGPT 免费版就能用（有每日次数限制） |
| 🎨 **能画什么** | 几乎任何东西：风景、人物、卡通、抽象艺术…… |

> 💡 **重点**：你不需要打开任何其他网站或 APP，**直接在 ChatGPT 对话框里打字就行**。

## 第三步：描述你想要的画面

在 ChatGPT 底部的输入框中，用**中文或英文**描述你想要的画面。

### 试试这样写：

> 画一只戴着墨镜的小猫，坐在沙滩上喝椰汁，卡通风格

然后按 **Enter 键**发送。

就这么简单！ChatGPT 会自动调用 DALL·E 来为你生成图片。

> 💡 **提示**：描述越具体，画出来的图越接近你想要的。只说"画一只猫"可能出乎你意料，但说"画一只橘色的胖猫，躺在阳光下的窗台上打盹，日系水彩风格"就精确多了。

### 几个例子，你可以直接复制试试：

| 描述 | 风格 |
|------|------|
| 一只柴犬穿着西装，在办公室里认真工作，搞笑卡通风格 | 🐕 搞笑卡通 |
| 中国传统水墨画风格的山水，远处有飞鸟 | 🏔️ 水墨国风 |
| 一个女孩撑着油纸伞走在雨中的小巷，动漫风格 | 🌧️ 日系动漫 |
| 未来城市夜景，霓虹灯倒映在雨水中，赛博朋克风格 | 🌃 赛博朋克 |

## 第五步：尝试修改和变化

对结果不满意？没关系！你可以继续在对话中告诉 ChatGPT 怎么调整：

### 常用的调整话术：

| 你想调整的 | 可以这样说 |
|-----------|-----------|
| 换个风格 | "换成水彩画风格" |
| 换个角度 | "从俯视角度看" |
| 加点细节 | "再加上飘落的樱花" |
| 换个色调 | "用暖色调，日落的氛围" |
| 更可爱 | "让它更萌一些，大眼睛" |
| 再画一张 | "再生成一张不一样的" |

ChatGPT 会记住你之前说了什么，所以你只需要说**变化的部分**就行。

> 💡 **小技巧**：如果想从头开始，点击左侧的 **"New chat"** 开一个新对话，这样之前的描述不会影响新的图片。

## 免费版有什么限制？

ChatGPT 免费版使用 DALL·E 有一些限制：

| 限制项 | 说明 |
|--------|------|
| 📊 **每日次数** | 免费版每天可生成的图片数量有限（具体数量 OpenAI 会调整） |
| 🖼️ **每次数量** | 通常一次生成 1 张图片 |
| ⏱️ **速度** | 高峰期可能需要排队等待 |
| 📐 **尺寸** | 可选正方形、横版、竖版 |

> 💡 **够用吗？** 对于日常使用和个人爱好，免费版完全够用。如果你是做设计工作需要大量出图，可以考虑升级 ChatGPT Plus 或试试 Midjourney。

## 常见问题

<FAQItem question="DALL·E 和 ChatGPT 是什么关系？">
  DALL·E 是 OpenAI 开发的图片生成模型，它已经<strong>内置</strong>在 ChatGPT 里了。你不需要单独打开 DALL·E，直接在 ChatGPT 对话框里描述画面，ChatGPT 就会自动调用 DALL·E 来画图。
</FAQItem>

<FAQItem question="只能画英文吗？">
  <strong>不！可以直接用中文描述。</strong>DALL·E 支持中文，你用中文写描述，它就能理解并画出来。不过有时候英文描述的效果会更精确一些，因为这个模型是基于英文训练的。
</FAQItem>

<FAQItem question="生成的图片版权归谁？">
  根据 OpenAI 的政策，DALL·E 生成的图片版权归<strong>使用者（也就是你）</strong>所有。你可以自由使用，包括商业用途。但如果图片中包含真实的名人或品牌，使用时要注意。
</FAQItem>

<FAQItem question="为什么画出来的人脸有时很奇怪？">
  AI 画人脸一直是技术难点，特别是手指、耳朵这些细节。这是目前所有 AI 绘画工具的通病。多生成几次，总能找到满意的。也可以在描述中强调"高质量、精美细节"。
</FAQItem>

<FAQItem question="免费额度用完了怎么办？">
  等第二天重置就行。或者考虑升级 ChatGPT Plus（$20/月），获得更多生成次数。也可以用 <a href="/tutorials/ai-art/03-midjourney-setup">Midjourney</a> 作为替代方案。
</FAQItem>

<FAQItem question="手机上能用吗？">
  <strong>能！</strong>下载 ChatGPT 官方 APP（iOS / Android 都有），或者用手机浏览器访问 chatgpt.com，功能和电脑上一样。
</FAQItem>
