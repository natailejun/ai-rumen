## 第一步：理解提示词的万能结构

DALL·E 的提示词不是随便写的。一个好的提示词通常包含 **4 个要素**：

<Screenshot
  src="/images/tutorials/ai-art/prompt-structure.png"
  alt="提示词结构图示"
  caption="图：一个优秀提示词的四个组成部分"
/>

| 要素 | 说明 | 示例 |
|------|------|------|
| 🎯 **主体** | 画面的核心内容 | 一只柴犬、一个女孩、一座城市 |
| 🎨 **细节** | 具体的动作、表情、物品 | 戴着红色围巾、拿着雨伞、站在花田里 |
| 🖌️ **风格** | 艺术风格或画风 | 水彩画、赛博朋克、吉卜力风格 |
| 💡 **光线/氛围** | 光线和整体感觉 | 暖色调、金色夕阳、清晨薄雾 |

### 万能公式：

```
[主体] + [细节] + [风格] + [光线/氛围]
```

### 对比看看：

| ❌ 模糊的提示词 | ✅ 完整的提示词 |
|---|---|
| 画一只猫 | 一只胖胖的橘猫，坐在咖啡馆的窗台上，窗外下着小雨，日系水彩画风格，温暖的灯光 |
| 画一个女孩 | 一个短发女孩穿着白色连衣裙，站在薰衣草花田中回头看，吉卜力动画风格，夕阳余晖 |
| 画一座城市 | 未来城市夜景，霓虹灯招牌，飞行汽车穿梭在摩天大楼之间，赛博朋克风格，紫色和蓝色调 |

> 💡 **关键**：描述越具体，AI 越能"理解"你的想法。不要怕写太多，AI 能处理很长的描述。

## 第二步：掌握风格关键词

风格是决定图片"味道"的关键。以下是常用的风格词汇：

<Screenshot
  src="/images/tutorials/ai-art/style-comparison.png"
  alt="不同风格对比图"
  caption="图：同一个主体，不同风格关键词的效果对比"
/>

### 🎨 绘画/艺术风格

| 关键词 | 效果 | 适用场景 |
|--------|------|---------|
| 水彩画 / watercolor | 清透、柔和、有水渍感 | 文艺插图、绘本 |
| 油画 / oil painting | 厚重、色彩饱满 | 肖像、风景 |
| 素描 / pencil sketch | 黑白线条、手绘感 | 概念设计、文艺感 |
| 中国水墨画 / Chinese ink painting | 黑白为主、留白意境 | 中国传统题材 |
| 浮世绘 / ukiyo-e | 日本传统版画风格 | 日式题材 |
| 像素艺术 / pixel art | 复古游戏画面 | 游戏风格、怀旧感 |

### 📷 摄影/写实风格

| 关键词 | 效果 | 适用场景 |
|--------|------|---------|
| 写实照片 / photorealistic | 接近真实照片 | 逼真场景 |
| 电影感 / cinematic | 宽银幕、电影色调 | 故事感强的画面 |
| 航拍 / aerial view | 俯瞰视角 | 风景、城市 |
| 微距摄影 / macro photography | 超近距离、细节丰富 | 花草、昆虫 |

### 🌈 二次元/卡通风格

| 关键词 | 效果 | 适用场景 |
|--------|------|---------|
| 吉卜力风格 / Studio Ghibli style | 宫崎骏动画感 | 温馨治愈场景 |
| 日系动漫 / anime style | 日本动漫画风 | 角色、场景 |
| 皮克斯风格 / Pixar style | 3D 卡通、质感高级 | 趣味人物 |
| 迪士尼风格 / Disney style | 经典迪士尼动画 | 童话场景 |
| 扁平插画 / flat illustration | 简洁现代 | UI 设计、PPT 配图 |

### 🔮 特殊风格

| 关键词 | 效果 | 适用场景 |
|--------|------|---------|
| 赛博朋克 / cyberpunk | 霓虹灯、高科技、暗色调 | 未来城市 |
| 蒸汽朋克 / steampunk | 机械齿轮、维多利亚风 | 复古科幻 |
| 极简主义 / minimalist | 简洁、留白、干净 | 现代设计 |
| 超现实主义 / surrealism | 梦幻、不合理组合 | 创意艺术 |
| 像素游戏 / 8-bit game | 复古电子游戏 | 游戏相关 |

> 💡 **进阶技巧**：风格可以叠加！比如"水彩画风格 + 日系动漫"会产生水彩动漫的独特效果。

## 第三步：学习光线和氛围词

光线决定了图片的"情绪"：

| 关键词 | 效果 | 情绪 |
|--------|------|------|
| 暖色调 / warm tones | 偏黄偏橙 | 温馨、舒适 |
| 冷色调 / cool tones | 偏蓝偏紫 | 冷静、神秘 |
| 金色夕阳 / golden hour | 日落时的金色光线 | 浪漫、温暖 |
| 清晨薄雾 / morning mist | 柔和、朦胧 | 宁静、清新 |
| 霓虹灯光 / neon lights | 彩色光斑 | 活力、夜晚 |
| 逆光 / backlighting | 轮廓发光 | 唯美、梦幻 |
| 月光 / moonlight | 银色冷光 | 安静、神秘 |
| 戏剧性光线 / dramatic lighting | 强烈明暗对比 | 史诗、震撼 |

## 第四步：避免常见错误

<Screenshot
  src="/images/tutorials/ai-art/bad-vs-good-prompt.png"
  alt="错误提示词和正确提示词对比"
  caption="图：左图为模糊提示词的结果，右图为优化后的效果"
/>

### ❌ 常见错误清单

| 错误 | 问题 | ✅ 改进方法 |
|------|------|-----------|
| 描述太短 | "画一只狗" → AI 随机发挥 | 加上品种、颜色、动作、场景 |
| 一次性塞太多 | "画一只猫和一只狗和一个人和一座山和一条河" | 画面焦点控制在 1-2 个主体 |
| 描述矛盾 | "黑白的彩色画" | 确保各要素不冲突 |
| 没有指定风格 | AI 默认随机选风格 | 明确写出想要的风格 |
| 期望照片级真实 | 但描述用"卡通"风格 | 风格和期望要匹配 |

<WarningCallout title="关于人物手指" type="warning">
  AI 画人物手指是出了名的难。目前的 DALL·E 已经大幅改善，但偶尔还是会出错。
  <br /><br />
  <strong>建议</strong>：如果画面中手指很重要，可以加上 "correct hands, detailed fingers" 这样的强调词。或者选择不露手的构图。
</WarningCallout>

## 第五步：实战练习

光看不练假把式！跟着下面的练习动手试试：

### 练习 1：基础补全（5 分钟）

用下面的万能公式，补全这个提示词：

> 一只 ______ 的 ______，在 ______ 地方，______ 风格，______ 光线

写出你的版本，然后在 ChatGPT 中试试，看看效果如何。

### 练习 2：风格对比（5 分钟）

用**同一个主体**，分别用以下 3 种风格生成图片：

1. "一只柴犬，写实摄影风格"
2. "一只柴犬，吉卜力动画风格"
3. "一只柴犬，赛博朋克风格"

对比三张图的差异，感受风格关键词的威力。

### 练习 3：从模糊到精确（5 分钟）

先用模糊描述生成一张图，然后逐步优化：

1. 第 1 次："画一个机器人"
2. 第 2 次："画一个可爱的圆滚滚的小机器人，站在森林里"
3. 第 3 次："画一个可爱的圆滚滚的小机器人，大大的眼睛，站在秋天的森林里，落叶飘落，温暖的夕阳光线，皮克斯 3D 动画风格"

对比三次的结果，你会惊讶于差距有多大。

### 练习 4：改写真实场景（3 分钟）

描述你现在所在的房间，用提示词公式写一段描述，让 AI 画出来。试试看 AI 能还原多少？

### 练习 5：创造不存在的东西（2 分钟）

这是最有趣的部分——用 AI 画出世界上不存在的东西：

> "一只由彩色水晶构成的鹿，鹿角上开着发光的花朵，站在月球表面，地球在背景中，超现实主义风格"

发挥你的想象力，越天马行空越好！

## 常见问题

<FAQItem question="提示词用中文还是英文效果好？">
  都可以，但<strong>英文通常效果更好</strong>。因为 DALL·E 的训练数据以英文为主，英文提示词的细节表达更精确。如果你英文不太好，可以用中文先写，然后让 ChatGPT 帮你翻译成英文再生成。
</FAQItem>

<FAQItem question="提示词写多长合适？">
  建议 <strong>30-100 个词</strong>（英文）或 <strong>50-200 个字</strong>（中文）。太短信息不够，太长 AI 可能抓不住重点。关键是把最重要的信息放在前面。
</FAQItem>

<FAQItem question="为什么我照着模板写，效果还是不好？">
  AI 绘画有随机性，同一个提示词每次生成的图都不一样。多生成几次（通常 3-5 次）总能找到满意的。另外，有时候微妙的措辞变化会带来截然不同的效果，多尝试不同的表达方式。
</FAQItem>

<FAQItem question="能用别人的图作为参考吗？">
  DALL·E 本身不支持"以图生图"。但你可以上传一张图片到 ChatGPT，然后描述你想怎么修改，ChatGPT 会理解图片内容并根据你的要求生成新的图片。
</FAQItem>
