## 本教程目标

学完这篇，你会知道：

- 根据自己的需求选择最合适的 AI 工具
- 了解每个工具的优缺点
- 避免选择困难和踩坑

---

## 按需求选择

### 场景一：我就是想先试试 AI 是什么感觉

👉 **推荐：DeepSeek**

- 免费，不需要科学上网
- 中文能力非常好
- 网页版直接打开就能用
- 访问：[chat.deepseek.com](https://chat.deepseek.com)

### 场景二：我想用 AI 提高工作效率（写周报、润色文案、翻译）

👉 **推荐：Kimi 或 ChatGPT**

- **Kimi**：国产，免费，中文好，能处理超长文档。适合处理文件、查资料
- **ChatGPT**：综合能力最强，但需要科学上网。写作、翻译、头脑风暴都很出色

### 场景三：我想用 AI 写代码 / 学编程

👉 **推荐：Claude Code 或 Cursor**

- **Claude Code**：命令行工具，适合有一定编程基础的人。本网站的教程会带你从零开始
- **Cursor**：AI 增强的代码编辑器，和 VS Code 几乎一样，更直观

### 场景四：我想找个免费的国产 AI

👉 **推荐：DeepSeek、Kimi、豆包、通义千问**

这些都不需要科学上网，注册简单，各有特色：

| 工具 | 最大优势 | 免费额度 |
|------|---------|---------|
| DeepSeek | 推理能力强，接近 GPT-4 | 完全免费 |
| Kimi | 200K 超长上下文 | 免费 |
| 豆包 | 字节出品，语音交互好 | 免费 |
| 通义千问 | 阿里出品，办公场景好 | 免费 |

## 简单决策树

```
你现在最想用 AI 做什么？
│
├─ 聊天、写东西 → 能科学上网吗？
│   ├─ 能 → 用 ChatGPT
│   └─ 不能 → 用 DeepSeek 或 Kimi
│
├─ 写代码 → 有编程基础吗？
│   ├─ 有一点 → 用 Claude Code
│   └─ 基本没有 → 用 Cursor
│
├─ 处理长文档 → 用 Kimi
│
└─ 就试试看 → 用 DeepSeek（最快上手）
```

---

## 我的建议：先学哪个？

如果你是**完全零基础**，我建议按这个顺序：

1. **先学 DeepSeek**：免费、无需特殊网络、中文好 → 建立信心
2. **再学 ChatGPT**（如果你能科学上网）：综合能力最强 → 体验顶级 AI
3. **然后学 Kimi**：处理文档、长文本 → 工作效率翻倍
4. **最后学 Claude Code/Cursor**：如果你想用 AI 编程

## 常见问题

<FAQItem question="我只能选一个，推荐哪个？">
  如果你能科学上网，选 <strong>ChatGPT</strong>。如果不能，选 <strong>DeepSeek</strong>。这两个是综合能力最强的免费选择。
</FAQItem>

<FAQItem question="一定要用外国的吗？国产的够不够好？">
  2026 年的国产 AI 已经非常强了。DeepSeek 的推理能力接近 GPT-4，Kimi 处理长文档的能力甚至超过 ChatGPT。对于绝大多数日常需求，国产工具完全够用。
</FAQItem>

<FAQItem question="免费版和付费版差多少？">
  对初学者来说，免费版完全够用。付费版通常在速度、使用量、最新功能上有优势，但你暂时不需要。
</FAQItem>
