# 💻 Cursor IDE 从入门到精通

**Cursor** 是基于 VS Code 的 AI 增强代码编辑器。如果你用过 VS Code，界面几乎一模一样——只是多了 AI 功能。

---

## 本系列教程包含

### [⬇️ 下载与安装 Cursor](/tutorials/cursor/01-download-install)
下载安装包，完成首次配置，从 VS Code 迁移设置。

### [✏️ 第一次 AI 编辑](/tutorials/cursor/02-first-edit)
体验 Cursor 核心功能——用 AI 写代码、改代码、生成完整文件。

### [💬 Chat 与 Composer](/tutorials/cursor/03-chat-composer)
深入 Cursor 的三大 AI 功能：Chat（对话）、Ctrl+K（行内编辑）、Composer（多文件生成）。

### [🚀 实用技巧合集](/tutorials/cursor/04-tips-tricks)
Rules for AI、Privacy Mode、模型选择、快捷键——让你的 Cursor 效率翻倍。

## Cursor vs Claude Code vs VS Code

| 对比 | Cursor | Claude Code | VS Code |
|------|:---:|:---:|:---:|
| 界面 | 图形界面 | 命令行 | 图形界面 |
| AI 功能 | ✅ 内置 | ✅ 内置 | ❌ 需装插件 |
| VS Code 扩展 | ✅ 兼容 | ❌ | ✅ 原生 |
| 学习曲线 | 低（和 VS Code 一样） | 中（命令行） | 低 |
| 付费 | $20/月 | 按量/对话 | 免费 |
