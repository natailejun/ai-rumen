## 本教程目标

获取 DeepSeek API Key，用代码调用 DeepSeek。

---

## 获取 API Key

1. 访问 [**platform.deepseek.com**](https://platform.deepseek.com)
2. 登录后在左侧找到 **API Keys**
3. 点击 **创建 API Key**
4. 复制保存

<WarningCallout title="提醒">
  API Key 不要分享给他人，也不要上传到公开仓库。
</WarningCallout>

---

## Python 调用示例

```python
import requests

response = requests.post(
    "https://api.deepseek.com/v1/chat/completions",
    headers={"Authorization": "Bearer 你的API_Key"},
    json={
        "model": "deepseek-chat",
        "messages": [{"role": "user", "content": "你好"}]
    }
)
print(response.json())
```
