## 本教程目标

学完这篇，你会掌握：

- ✅ 写出有效的提示词（Prompt）的基本方法

- ✅ 知道什么样的提问方式能得到好答案
- ✅ 学会多轮对话——像聊天一样不断优化结果

## 核心概念：什么是「提示词」？

**提示词（Prompt）** 就是你跟 AI 说的话。你的问题写得好，AI 的回答就好；问题写得模糊，回答也模糊。

> 坏的提示词："帮我写点东西"
> 好的提示词："帮我写一封得体的请假邮件，我是市场部的，因为明天要去看牙医需要请半天假，语气要正式但不生硬，100 字左右"

看出区别了吗？**好提示词的秘诀就是：具体、具体、再具体**。

---

## 黄金公式：好的提示词 = 角色 + 任务 + 要求

```
好的提示词 = [你希望 AI 扮演什么角色?] + [要完成什么任务?] + [有什么具体要求?]
```

### 例子 1：写周报

❌ **不好的写法：**
> "帮我写周报"

✅ **好的写法：**
> "你是一个互联网公司的产品经理。本周围绕用户增长做了 A/B 测试，优化了注册流程，转化率从 5% 提升到 8%。帮我写一份简洁的周报，列出本周工作、下周计划、遇到的问题。200 字以内，用分点呈现。"

---

### 例子 2：润色文案

❌ **不好的写法：**
> "这段改好看点"

✅ **好的写法：**
> "下面是一段给客户的邮件草稿，请帮我润色，让它更正式、更礼貌，同时保持简洁。去掉口语化的表达，但不要变得很生硬。"

---

### 例子 3：翻译

❌ **不好的写法：**
> "翻成英文"

✅ **好的写法：**
> "请把下面这段中文翻译成英文，要求商务正式风格，适合用在产品发布会上。保持专业术语准确。"

## 提示词模板（可以直接用）

### 📝 写邮件模板

```
你是一个[你的职位]的职场人士。
请帮我写一封邮件，收件人是[谁]，主题是[什么]。
邮件需要[正式/轻松/礼貌]的语气。
[特殊要求：如字数、格式、附件提醒等]
```

### 📊 写周报模板

```
请帮我写一份周报。
我的职位是[职位]，本周做了以下工作：
1. [工作1]
2. [工作2]
3. [工作3]

请用简洁的方式呈现，包括"本周完成"、"遇到问题"、"下周计划"三个部分。
```

### ✨ 润色文案模板

```
请帮我润色下面这段文字。
要求：
- [更正式 / 更口语 / 更有说服力]
- [保留原意 / 可以适当改写]
- [字数要求]

[贴上你需要润色的文字]
```

## 进阶技巧：多轮对话

ChatGPT 最强大的不是"一次提问"，而是**持续对话**。

### 例子：从粗糙到精致

<Screenshot
  src="/images/tutorials/chatgpt/multi-turn-example.png"
  alt="多轮对话示例截图"
  caption="图：通过多轮对话不断优化 AI 的输出"
/>

```
你：帮我写一个减肥计划

AI：[给出一个通用计划]

你：我是 30 岁女性，办公室久坐工作，没有时间去健身房。
    一周能运动 2 次，每次半小时。重新写一下。

AI：[针对久坐女性的计划]

你：我不喜欢跑步，有没有别的方式？

AI：[替换跑步为跳绳、HIIT 等]

你：加一个饮食建议的板块，控制在 1500 卡左右。

AI：[加入饮食建议的完整计划]
```

**关键点**：
1. 第一轮不用写太完美——给 AI 一个基本框架
2. 第二轮加细节和约束
3. 第三轮修正你不满意的地方
4. 第四轮加新要求

> 💡 把 ChatGPT 当作一个**随时可以提修改意见的助理**，而不是一个"下命令就拿结果"的工具。

## 不同场景的提示词示例

### 学习与理解
```
请用初中生能听懂的语言解释什么是机器学习。
用生活中的例子说明，不要用专业术语。
```

### 头脑风暴
```
我在做一个关于环保的创业项目，需要 5 个有创意的产品名字。
要求：好记、有意义、不超过 4 个字。
每个名字附带一句话说明。
```

### 代码辅助
```
用 Python 写一个简单的待办事项程序。
要求：
- 能在命令行运行
- 支持添加、查看、删除任务
- 有注释说明每个部分的功能
```

### 数据分析（需要上传文件）
```
我上传了一份销售数据。
请帮我分析：
1. 哪个月份销售额最高
2. 前 5 名产品是什么
3. 给出 3 条提升销售额的建议
```

## 实用小技巧

| 技巧 | 说明 |
|------|------|
| **说"继续"** | 如果回答太长被截断，输入"继续"，AI 会接着往下写 |
| **说"重写，这次更...一些"** | 不满意就直接提修改意见，不要重新问 |
| **给示例** | 先说"我想要这种风格：[贴一个例子]"，效果拔群 |
| **设字数限制** | "控制在 200 字以内"可以避免 AI 废话太多 |
| **要 AI 解释自己** | "你为什么这么写？"可以让 AI 解释它的思路 |

## 常见问题

<FAQItem question="为什么 ChatGPT 的回答有时不对？">
  AI 会产生"幻觉"——编造看似合理但错误的信息。遇到重要内容（数据、日期、人名），最好自己再验证一下。
</FAQItem>

<FAQItem question="ChatGPT 会记得我在其他对话中说过的话吗？">
  不会。每个对话是独立的。如果你想让它"记住"一些信息，可以在当前对话中直接告诉它。
</FAQItem>

<FAQItem question="ChatGPT 能用中文写代码注释吗？">
  完全可以。"请给这段代码加上详细的中文注释"，它会照做。
</FAQItem>
