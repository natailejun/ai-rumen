## 角色扮演法 —— 让 AI 变成专家

ChatGPT 最大的隐藏技能：**它可以扮演任何人**。

### 为什么角色扮演有用？

当你说"帮我分析这段代码"，AI 给的是通用回答。
当你说"你是一个有 10 年经验的 Java 架构师，请帮我分析这段代码"，AI 给的回答专业度完全不同。

### 经典角色清单

| 角色 | 提示词示例 |
|------|-----------|
| **写作教练** | "你是一个专业写作教练，帮我改进这篇文章的..." |
| **代码审查员** | "你是一个资深后端工程师。审查下面的代码，指出潜在问题..." |
| **面试官** | "你是腾讯的产品经理面试官。对我进行一次模拟面试..." |
| **英语老师** | "你是我的私人英语老师。和我用英语对话，纠正我的语法错误..." |
| **理财顾问** | "你是一个中立的理财规划师。根据我的情况给出建议..." |
| **法律顾问** | "你是一个熟悉中国劳动法的律师。分析以下情况..." |

> ⚠️ **注意**：AI 扮演律师、医生等角色仅供参考，不构成真正的法律/医疗建议。

## 自定义指令（Custom Instructions）—— 让 ChatGPT 记住你

<Screenshot
  src="/images/tutorials/chatgpt/custom-instructions.png"
  alt="自定义指令设置页面截图"
  caption="图：在 Settings → Customize ChatGPT 中设置你的偏好"
/>

**位置**：侧边栏 → 你的头像 → **Customize ChatGPT**

自定义指令会在**你每次对话时自动生效**，不需要每次都告诉 AI。

### 怎么写自定义指令？

ChatGPT 会问两个问题：

**问题 1：你希望 ChatGPT 了解你的什么信息？**

```
我在北京工作，30岁，互联网行业产品经理。
日常大量需要写周报、产品需求文档、竞品分析报告。
中文母语，英语能阅读但写作需要帮助。
```

**问题 2：你希望 ChatGPT 怎么回复你？**

```
- 用简体中文回复
- 回答简洁直接，不超过 300 字
- 需要列表的时候用分点格式
- 如果不确定，直接说"我不确定"，不要编造
```

### 自定义指令示例（全套）

```
【关于我】
- 职位：互联网公司市场部专员
- 日常任务：写营销文案、邮件、周报、数据分析
- 偏好：中文回复，简洁直接，用分点总结

【回复风格】
- 用简体中文
- 默认简洁模式，除非我要求详细
- 需要操作步骤时用编号
- 对比信息用表格呈现
- 不要用太学术的语言
```

> 💡 好的自定义指令能让 ChatGPT 的输出质量提升 50% 以上。花 5 分钟写一下，非常值得。

## GPTs —— 别人做好的专用 AI

<Screenshot
  src="/images/tutorials/chatgpt/gpts-store.png"

  caption="图：侧边栏 → Explore GPTs 进入 GPTs 商店"

GPTs 是封装好的专用 AI 助手，你不需要写提示词，直接用就行。

### 值得一试的免费 GPTs

| GPTs 名称 | 功能 | 适合谁 |
|-----------|------|--------|
| **DALL·E** | AI 画图 | 所有人 |
| **Data Analyst** | 上传表格做分析 | 职场人士 |
| **Web Browser** | 联网搜索 | 所有人 |
| **Writing Coach** | 写作教练 | 学生、职场 |
| **Code Copilot** | 编程助手 | 开发者 |
| **Creative Writing Coach** | 创意写作 | 写作者 |
| **Math Solver** | 数学问题 | 学生 |

### 怎么用 GPTs？

1. 在侧边栏点击 **Explore GPTs**
2. 搜索你需要类型的 GPTs
3. 点击进入，会打开一个专门的对话窗口
4. 直接对话即可——GPTs 已经预设了行为和知识

> 💡 **Plus 用户**可以自己创建 GPTs，把你的常用提示词"打包"成一个专属 GPTs。

ChatGPT（GPT-4o）支持上传文件并分析：

### 支持的文件类型

| 类型 | 示例 |
|------|------|
| **文档** | PDF、Word、TXT |
| **表格** | Excel、CSV |
| **图片** | PNG、JPG、GIF |
| **代码** | Python、JS、HTML 等 |

### 实用场景

| 场景 | 操作 |

| 提取 PDF 要点 | 上传 PDF → "总结这份报告的 3 个核心观点" |

| 读懂截图 | 上传截图 → "这个报错是什么意思？" |
| 翻译文档 | 上传 PDF → "把这份英文 PDF 翻译成中文" |

## 进阶提示词模板合集

### 模板 1：角色扮演 + 任务 + 输出格式
```
你是一个[某领域专家]。
请[要做的事情]。
用[格式：表格/分点/段落]呈现，[字数/其他要求]。
```

### 模板 2：对比分析
```
请对比 [A] 和 [B]，从以下维度分析：
1. [维度1]
2. [维度2]
3. [维度3]

用表格呈现，并在最后给出你的建议。
```

### 模板 3：逐步拆解
```
我想学习 [某个技能]，但我是零基础。
请把学习路径拆成 5 个步骤，每一步说明：
- 学什么
- 怎么学
- 学完能做什么

每一步推荐 1-2 个学习资源。
```

### 模板 4：决策辅助
```
我面临以下选择：[选项A] vs [选项B]
我的情况是：[简要描述]

请帮我分析每个选择的优缺点，用表格呈现。
最后给出你的推荐，说明理由。
```

## 常见问题

<FAQItem question="自定义指令和普通对话有什么区别？">
  自定义指令是<strong>全局生效</strong>的，每个新对话都会自动带上。普通对话只在当前会话中有效。把自定义指令当成你给 ChatGPT 的"个人档案"。
</FAQItem>

<FAQItem question="GPTs 和自己写提示词哪个好？">
  对于常见任务（写作、编程、数据分析），GPTs 已经预设得很好，不用重复写提示词。对于特殊需求，自己写提示词更灵活。
</FAQItem>

<FAQItem question="ChatGPT 会上传的文件泄露出去吗？">
  如果你在 Settings → Data controls 中关闭了"Improve the model for everyone"，OpenAI 不会用你的数据训练模型。但公司机密文件仍然不建议上传。
</FAQItem>
