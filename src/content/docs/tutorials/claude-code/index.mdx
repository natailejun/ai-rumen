# ⌨️ Claude Code 从入门到实战

**Claude Code** 是 Anthropic 推出的命令行 AI 编程工具。你可以在终端里直接用自然语言描述需求，Claude Code 会帮你写代码、改 bug、解释代码逻辑。

---

## 本系列教程包含

### [📦 安装 Node.js](/tutorials/claude-code/01-install-nodejs)
Claude Code 需要一个叫 Node.js 的环境。这篇手把手教你安装，Windows/Mac 都有详细说明。

### [⚡ 安装并配置 Claude Code](/tutorials/claude-code/02-install-claude)
获取 Anthropic API Key，用 npm 安装 Claude Code CLI，完成初始配置。

### [🚀 第一个 AI 编程项目](/tutorials/claude-code/03-first-project)
用 Claude Code 创建你的第一个程序——从"新建项目"到"看到运行结果"，全程不超过 20 分钟。

### [💼 日常使用工作流](/tutorials/claude-code/04-workflow)
如何在日常开发中高效使用 Claude Code？文件操作、代码修改、Git 协作、成本控制等实用技巧。

### [🧠 高级功能与进阶技巧](/tutorials/claude-code/05-advanced)
自定义配置、MCP 工具扩展、批量任务、IDE 集成等进阶玩法。

---

## 学完你会什么？

- 在命令行中用 AI 创建完整项目
- 让 AI 帮你读懂别人的代码
- 用一句话重构几百行代码
- 理解 AI 编程的边界和最佳实践

> 💡 **前置技能**：不需要会编程！但最好对命令行有基本了解（知道 `cd`、`ls` 等基础命令）。零基础也能跟着做。
