# 📖 AI 术语表

用最通俗的语言解释常见的 AI 术语。

---

## A — C

| 术语 | 英文 | 大白话解释 |
|------|------|-----------|
| **AI（人工智能）** | Artificial Intelligence | 让计算机模仿人类智能的技术统称 |
| **API** | Application Programming Interface | 程序之间"对话"的接口。就像餐厅的菜单——你点菜（发请求），厨房做菜（处理），上菜（返回结果） |
| **API Key** | API Key | 使用 API 的"钥匙"，一串密码。**不要分享给别人** |
| **ChatGPT** | — | OpenAI 出的 AI 对话工具，目前最有名的 AI 产品 |
| **Claude** | — | Anthropic 出的 AI 助手，编程和安全领域表现出色 |
| **CLI** | Command Line Interface | 命令行界面，黑底白字的那个东西 |
| **上下文（Context）** | Context | AI 在一段对话中能"记住"的内容。上下文越长，能处理的东西越多 |

---

## D — G

| 术语 | 英文 | 大白话解释 |
|------|------|-----------|
| **大语言模型（LLM）** | Large Language Model | 就是"AI"的学名。ChatGPT、Claude、DeepSeek 都是大语言模型 |
| **GPT** | Generative Pre-trained Transformer | OpenAI 开发的一系列 AI 模型的名字 |
| **GPTs** | — | ChatGPT 里别人做好的专用 AI 助手，像"AI 小程序" |

---

## H — M

| 术语 | 英文 | 大白话解释 |
|------|------|-----------|
| **幻觉（Hallucination）** | Hallucination | AI 编造信息。"一本正经地胡说八道"。遇到重要信息要自己验证 |
| **MCP** | Model Context Protocol | 让 AI 连接外部工具/数据库的协议标准 |
| **MD/MDX** | Markdown / MDX | 一种简单的文本格式，用来写文档。本网站就是用 Markdown 写的 |
| **机器学习** | Machine Learning | AI 的一个分支，让计算机从数据中"学习" |

---

## N — P

| 术语 | 英文 | 大白话解释 |
|------|------|-----------|
| **npm** | Node Package Manager | Node.js 的"应用商店"，用来安装各种工具 |
| **Node.js** | — | 一个运行环境，很多编程工具依赖它 |
| **OpenAI** | — | 开发 ChatGPT 的公司 |
| **提示词（Prompt）** | Prompt | 你跟 AI 说的话。提示词写得好，回答就好 |

---

## R — Z

| 术语 | 英文 | 大白话解释 |
|------|------|-----------|
| **R1** | — | DeepSeek 的推理模式，会"边想边说" |
| **Token** | Token | AI 处理文本的最小单位。一个中文字 ≈ 1-2 个 token。计费按 token 算 |
| **模型（Model）** | Model | AI 的"大脑"。不同模型能力不同，就像不同水平的助手 |
| **微调（Fine-tuning）** | Fine-tuning | 在现有 AI 的基础上针对特定任务做额外训练 |
